# Shopify-Developer-Test
A custom Shopify Online Store 2.0 homepage built as part of a Shopify Developer assessment. The implementation focuses on editability, responsive design, and clean section architecture using native Shopify sections, blocks, Liquid, HTML, CSS, and lightweight JavaScript without relying on third-party page builders or paid apps.

Features
Custom Hero section with editable headings, buttons, and image
Configurable Benefits section with reorderable blocks
Featured Products section using Shopify product selections or manual product cards
Collections showcase with editable collection cards and optional slider functionality
Customer Reviews section with ratings, reviewer details, and optional product references
Editable Footer with menu groups, contact information, social links, and copyright content
Fully responsive across desktop, tablet, and mobile devices
Theme Editor friendly with all visible content manageable by non-technical users
Tech Stack
Shopify Online Store 2.0
Liquid
HTML
CSS
JavaScript
Project Goal

The objective of this assessment was to build a practical, maintainable, and merchant-friendly homepage experience while following Shopify best practices and avoiding unnecessary complexity.
